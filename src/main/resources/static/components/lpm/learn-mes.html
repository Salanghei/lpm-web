学习风格

相关描述：学习风格分析是针对学习者的学习数据统计日志、评论等数据进行分析建模得到的，采用五因素性格模型进行定义，将学习风格划分为五个维度，即外倾性、神经质、宜人性、经验开放性和严谨性。

标签描述
标签 描述
宜人型 主要判断是否友善。衡量一个人的信任和乐于助人的性质，以及一个人是否普遍脾气暴躁
严谨型 描述对事情的处理的细节。具有高度责任感特质的人倾向于有组织和可靠，表现出自律，尽职尽责
外倾型 主要是判断内向性和外向性。得分高的人性格开放，善于交流。得分低的人，相对比较冷漠、保守和害羞
神经质 主要是判断人情绪的稳定性。得分高的人容易产生不愉快情绪的倾向。得分低的人性格相对稳重，但可能缺乏上进心
经验开放型 描述对新生事物的态度。开放性反映了学习者的好奇心，创造力以及对一个人的新奇和多样性的偏好

学习兴趣

学习兴趣挖掘是基于学习者已有行为信息，包括浏览历史、搜索记录、点击信息等归纳出学习者兴趣偏好的描述。本系统中将学习者的兴趣特征分成三个层级表示：感兴趣的知识主题、感兴趣的课程、感兴趣的领域。学习兴趣的挖掘，是后续实现个性化课程推荐的基础。
